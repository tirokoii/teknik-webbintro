HTML

HTML som står för HyperText Markup Language är ett märkspråk som huvudsaklingen andvänds till att skapa webbsidor. Denna sida är t.ex skapad med HTML. Språket innehåller taggar som ska beskriva hur det ska se ut några exempel är:

  • div som fungerar som boxar eller avdelningar
  • li som skapar punkterade listor
  • h1 största rubriken
  • p paragraf eller brödtext

    • CSS

    CSS som står för Cascading Style Sheet är ett stillmallsspråk som märkspråk som HTML använder för att skapa stilregler. Vilket gör det enklare att redigera din HTML kod. CSS andvänder sig av style och class för att göra ändringar i HTML dokumentet.
